Meus Projetos

Projeto Brasa Burguer

O Brasa Burguer é um projeto desenvolvido para simular a experiência de um restaurante moderno de hambúrgueres artesanais, unindo sabor, praticidade e tecnologia. O site foi estruturado em HTML para garantir semântica e organização, estilizado com CSS e Bootstrap para oferecer uma interface acessível, responsiva e visualmente atraente, além de contar com JavaScript puro para funcionalidades dinâmicas, como o gerenciamento do carrinho de compras. O objetivo é proporcionar uma navegação intuitiva, que reflita a identidade de uma hamburgueria autêntica e aconchegante.

calculadora

Este projeto consiste em uma calculadora desenvolvida utilizando HTML, CSS e JavaScript puro, com foco em oferecer uma experiência prática e intuitiva ao usuário. A interface foi cuidadosamente planejada, com botões dispostos de maneira organizada para facilitar operações básicas como adição, subtração, multiplicação e divisão. O JavaScript é responsável por gerenciar os cálculos em tempo real, garantindo que os resultados sejam exibidos imediatamente à medida que o usuário interage com a aplicação. Além disso, a calculadora possui um design responsivo.

Projeto Alura

Esta capa representa a interface de uma página promocional para o Combo+ da Alura, que reúne os serviços do Alura+ e do Alura Língua em um único plano. O design foi cuidadosamente elaborado para transmitir diversidade e conexão com os usuários, destacando rostos de diferentes pessoas que reforçam a ideia de inclusão e identificação. Os elementos visuais, como imagens, cores e tipografia, foram organizados de forma a criar uma hierarquia clara, facilitando a navegação e destacando as informações mais importantes.Os botões de ação possuem cores chamativas.
